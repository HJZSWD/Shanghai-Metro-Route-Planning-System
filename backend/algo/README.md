# 算法文档

## Dijkstra 双主键最短路径

**文件**：`backend/src/pathfinder.cpp`

支持时间优先和换乘优先两种搜索模式，通过 `primaryIsTime` 参数切换主关键字。

```
时间优先：(主) 累计时间 → (次) 换乘次数
换乘优先：(主) 换乘次数 → (次) 累计时间
```

**剪枝策略**：
1. 避免回路：`nextId ≠ prev[curId]`
2. 阻断边过滤（Yen's 算法使用）
3. 禁止节点过滤（Yen's 算法使用）
4. 关闭站点过滤（通过 `getAvailableEdges`）

## Yen's K短路算法

用于生成前 K 条候选路径（TOP3），替代旧的单一边阻断法。

**原理**：
1. 先求第 1 条最短路径（Dijkstra）
2. 对每条已选路径的每个节点做"偏差"：
   - 根路径：从起点到当前节点
   - 阻断边：阻断所有已选路径中与根路径前缀重合的出边
   - 禁止节点：根路径中除偏差节点外的所有节点不可重复
3. 求偏差节点到终点的最短路径，合并为完整候选路径
4. 从全局候选堆中取最优作为下一条最终路径

**优势**：保证找到全局前 K 条最短路，路径多样性高，遗漏率为零。

## 路径还原

Dijkstra 搜索完成后，通过前驱数组回溯得到完整路径，再遍历路径计算总耗时和换乘信息。

**换乘判定**：相邻两段边所属线路不同，或边类型为"换乘"，则记为一次换乘。
